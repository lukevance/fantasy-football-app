### Usage

#### Resources

*/scoring-leaders*

Returns the top 50 scoring leaders for a given week and position.
Query params: 
	week - integer
	position - [QB, RB, WR, TE, FX, DST, K]
	

*/teams*

Returns all teams in a give league.
Query params:
	* leagueId - integer

*/players*

Returns players in alphabetical order.
